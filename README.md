# Student Management System - Arid Agriculture University Portal

A professional-grade, full-stack web application designed to manage student records efficiently. This platform enables administrators to seamlessly perform CRUD (Create, Read, Update, Delete) operations on student data, ensuring a streamlined administrative workflow.

## 📌 Project Overview
This Student Management System serves as a centralized portal for managing academic information. Built with a Node.js/Express backend and a responsive, modern frontend, it features a clean UI/UX with a gradient-based design theme, ensuring both performance and aesthetic appeal.

## ✨ Key Features
- **Student Data Management:** Easily add new student entries with details like Name, Arid Number, Degree, and CGPA.
- **Dynamic Dashboard:** Real-time viewing of student records in a responsive table format.
- **Update & Delete:** Modify existing records or remove students from the system with simple, intuitive controls.
- **Modern UI:** Built with Bootstrap and custom CSS, featuring glassmorphism elements, gradient color schemes, and a mobile-responsive layout.
- **Robust Backend:** Powered by Express.js to handle API requests and data processing.

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js
- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5
- **Icons & Fonts:** FontAwesome, Google Fonts (Inter)
- **Deployment:** Render
